# Changelog

All notable changes to the AV.Eases package will be documented in this file.

## [1.0.0] - 2025-01-13

### Added
- Initial release
- EEase enum with 30+ easing functions
- EaseConfig ScriptableObject
- EaseEvaluator for evaluating easing functions
- Linear, Sine, Quad, Cubic, Quart, Quint, Expo, Circ, Back, Elastic, Bounce variants
- In, Out, and InOut variants for each ease type
- MIT License
- Package publishing setup
